# stringUtils for Mad Pascal

Set of useful functions for string manipulation in TeBe's [Mad Pascal](https://github.com/tebe6502/Mad-Pascal).

Also take a look at Bocianu's [blibs](https://gitlab.com/bocianu/blibs).
